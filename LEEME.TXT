Modificado por: Luisana Zambrano

Probado en impresoras:
Bixolon SRP350
Biloxon 270
Biloxon 350 
Aclas PP1F3
Dascom Tally 1125
Star HSP7000

Funciona en sistemas Windows y Linux

Para correr el proyecto se necesita:
 -Python 2.5 o mayor
 -Libreria python serial (pySerial)

Para correr el ejemplo de una factura
basta con el siguiente comando

   python factfis.py ejemplo.prm

Automaticamente puede detectar el puerto
al cual ha sido conectada la impresora, 
tanto en Linux como en Windows.
